## Description

A price emitter.

## Installation

```bash
$ npm install
```

## Configuration

Create a `.env.local` file in the main project directory.
It needs to have a `PORT` value which will be used to connect to this service.

The file `config.json` contains the `type`, `tokens` that are served and `intervals` supported information.

## Running the app

```bash
$ npm run start

```
